# Entregas

Todas las entregas por parte de los alumnos deben hacerse en subdirectorios
de éste directorio. Los archivos deben generarse debajo de
`/entregas/<nombre_entrega>/<ApellidoNombre>/`. Por ejemplo, la entrega de
/Fulano de Tal/ para la primera consigna se entrega en
`/intereses/DeTalFulano/`.
